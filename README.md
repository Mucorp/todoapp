# Todo App (Larvel 8 - Vue 3 - Vuex 4)

### Install Local
- #### Clone Repo

	`git clone git@github.com:Mucorp/todoapp.git`

- #### Set up frontend
	- `cd frontend`
	
	- `npm install`
	
	- `npm run serve`
	
	- Update env with api url.
- #### Set up Backend
	- `cd backend`
  
  - update env with db credentials
  - `composer install`
  
  - `php artisan migrate`
  
  - `php artisan db:seed`
  
  - `php artisan serve`
